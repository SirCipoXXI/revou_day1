HTMl Layout Elemet

HTML memiliki beberapa element semantic yang menentukan berbagai bagia halaman web :
  • header - mendefinisikan header untuk document atau section
  • nav - mendefinisikan satu set link navigasi
  • section - mendefinisikan bagian dalam dokumen
  • article - mendefinisikan suatu konten article
  • aside - mendefinisikan bagian samping dari conten (seperti sidebar)
  • footer - mendefinisikan footer / bagian bawah dari dokumen atau section
  • details - mendefinisikan detail tambahan yang dapat dibuka dan ditutup penggunaan sesuai perintah
  • summary - mendefinisikan heading untuk element

HTML semantic Elemets

    non semantic Elemets (menceritakan apa-apa tentang isinya)
  • Div
  • span

    semantic Elemets (mendefinisikan isinya dengan jelas)
  • form
  • table
  • article

menuliskan element html dengan semantic yg tepat akan memudahkan proses development juga meningkatkan score SEO dan Accessebility web
