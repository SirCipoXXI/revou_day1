Beberapa cara atau metode untuk set color di CSS

  1. RGB atau red,green,blue color value

    Contoh : "color: RGB(0,0,225)"

  2. Hexadecimal value

    Contoh : "color: #00FF00"

  3. Hue, Saturation, Lightness value

    Contoh : "color: hsl(360,100%,75%)"

  4. Alpha Transparency

    Contoh : "color: RGB(0,0,225,0.782)"

    Contoh : "color: hsl(360,100%,75%,0.5)"

Display

Properti tampilan CSS yang mengatur apakah suatu element diperlukan sebagai block atau inline element dan tata letak yang digunakan untuk turunannya, seperti flow layout, grid, atau flex

  • Inline : menampilkan element sebagai elemen inline (seperti "span"). properti height dan width apapun tidak akan berpengaruh
  • Block : menampilkan element sebagai elemen block (seperti "p"). itu dimulai pada baris baru, dan menempati seluruh lebar
  • Flex : menampilkan element sebagai container yang flexibel
      • Contoh flex :
      Home
      Search
      Logout
  • grid : menampilkan element sebagai grid container

Width dan Hight, digunakan untuk mengatur lebar dan tinggi dari suatu element
